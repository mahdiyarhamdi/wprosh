خروجی گرفتن و آپدیت محصولات ووکامرس با فایل CSV به همراه گزارش خطای کامل و دقیق

== توضیحات ==

افزونه Wprosh یک ابزار حرفه‌ای برای مدیریت محصولات ووکامرس با استفاده از فایل CSV است.

= ویژگی‌ها =

* خروجی گرفتن از تمام محصولات ووکامرس (ساده، متغیر، واریاسیون)
* آپدیت محصولات با آپلود فایل CSV ویرایش شده
* گزارش خطای کامل و دقیق به زبان فارسی
* رابط کاربری مدرن و فارسی با فونت YekanBakh
* پشتیبانی از تمام فیلدهای استاندارد ووکامرس
* پشتیبانی از دسته‌بندی‌ها، برچسب‌ها و ویژگی‌ها

= فیلدهای قابل خروجی و آپدیت =

* شناسه، کد محصول (SKU)، نام، نامک
* نوع محصول، وضعیت انتشار
* توضیحات کامل و کوتاه
* قیمت اصلی و فروش ویژه
* تاریخ شروع و پایان تخفیف
* وضعیت و تعداد موجودی
* وزن و ابعاد
* دسته‌بندی‌ها و برچسب‌ها
* ویژگی‌ها
* و سایر فیلدها...

== نصب ==

1. پوشه `wprosh` را در مسیر `/wp-content/plugins/` آپلود کنید
2. از بخش افزونه‌ها در داشبورد وردپرس، افزونه را فعال کنید
3. از منوی داشبورد به صفحه "Wprosh" بروید

== نحوه استفاده ==

= خروجی گرفتن =

1. به صفحه Wprosh در داشبورد بروید
2. روی دکمه "خروجی گرفتن" کلیک کنید
3. فایل CSV به صورت خودکار دانلود می‌شود

= آپدیت محصولات =

1. فایل CSV خروجی را در اکسل یا نرم‌افزار مشابه باز کنید
2. تغییرات مورد نظر را انجام دهید (فیلدهای id، type و parent_id قابل تغییر نیستند)
3. فایل را ذخیره کنید
4. به صفحه Wprosh بروید
5. فایل را آپلود کنید و روی "آپدیت محصولات" کلیک کنید
6. گزارش خطاها در صورت وجود به صورت CSV دانلود می‌شود

= فرمت فیلدها =

* **دسته‌بندی‌ها و برچسب‌ها**: با | جدا کنید. مثال: `لباس|مردانه|پیراهن`
* **ویژگی‌ها**: فرمت JSON. مثال: `{"رنگ":"قرمز","سایز":"XL"}`
* **فیلدهای بولین**: از `yes`/`no` یا `1`/`0` استفاده کنید
* **تاریخ‌ها**: فرمت `YYYY-MM-DD`. مثال: `2024-12-31`
* **محصولات مرتبط**: شناسه‌ها را با | جدا کنید. مثال: `123|456|789`

== سوالات متداول ==

= آیا تصاویر محصولات در خروجی وجود دارد؟ =

خیر، به دلیل حجم بالا و پیچیدگی مدیریت تصاویر، فقط اطلاعات متنی محصولات خروجی گرفته می‌شود.

= چگونه محصولات با ID شناسایی می‌شوند؟ =

محصولات فقط با فیلد ID شناسایی می‌شوند. این فیلد باید در فایل CSV وجود داشته باشد و تغییر نکند.

= آیا می‌توانم محصول جدید اضافه کنم؟ =

خیر، این افزونه فقط برای ویرایش محصولات موجود طراحی شده است.

= چرا برخی فیلدها آپدیت نشدند؟ =

دلیل عدم آپدیت در فایل گزارش خطاها که پس از آپدیت دانلود می‌شود، توضیح داده شده است.
